Quelle est la base légale du traitement?

Info

La base légale d’un traitement est ce qui donne le droit à un organisme de traiter des données à caractère personnel.

Quelle est la finalité du traitement ?

Info

La finalité du traitement est l’objectif principal de l’utilisation de données personnelles. Les données de doivent pas être traitées ultérieurement de façon incompatible avec cet objectif initial. Si la finalité ne se trouve pas dans la liste, indiquez la dans la rubrique "Autre finalité".

Le traitement 1 manipulera t-il des données sensible ?

Info

Il s'agit de données sur l'origine raciale ou ethnique, les opinions politiques, la religion / la philosophie, l'appartenance à un syndicat, les caractéristiques génétiques, les caractéristiques biométriques, la santé, le comportement sexuel / l'orientation sexuelle / les données criminelles.
